# Industrial Training Kit

## Overview

This repository contains the projects developed during the DecodeLabs Industrial Training Program. It demonstrates practical implementation of frontend and backend development concepts using modern web technologies.

---

# Week 1 – Responsive Industrial Training Website

## Project Description

A responsive industrial training website developed using HTML, CSS, and JavaScript. The project focuses on creating a clean, user-friendly interface with responsive design principles.

## Features

- Responsive Layout
- Modern User Interface
- Navigation Menu
- Hero Section
- About Section
- Services Section
- Contact Section
- Interactive JavaScript Functionality

## Technologies

- HTML5
- CSS3
- JavaScript

# Week 2 – Spring Boot REST API

## Project Description

A RESTful web service developed using Spring Boot to understand backend development fundamentals and API implementation.

## Features

- REST API Development
- GET Endpoint
- POST Endpoint
- JSON Request and Response Handling
- Postman API Testing
- Maven Project Structure

## Technologies

- Java 21
- Spring Boot
- Maven
- REST API
- Postman
